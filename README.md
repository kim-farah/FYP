Kinderbot Learning Platform

A web-based educational platform that transforms physical Kinderbot learning materials into a digital experience for teachers, coordinators, and parents.

Features

Coordinator Dashboard
- Manage classes, teachers, parents, and activities
- Create and edit activities
- View recent activities taught by teachers
- Add/Edit/Delete classes and sections

Teacher Dashboard
- View assigned classes
- Start activities (adds to activity log)
- Assess students with ratings
- Send messages to parents

Parent Dashboard
- Track child's progress
- View completed activities
- Read teacher notes
- Read-only access

Technologies Used
- HTML5
- CSS3
- JavaScript
- Figma for UI/UX design
